# Serverless API

1. Create your project folder.
2. Inside the folder, ```sls create -t aws-nodejs```
3. Create features folder.
4. Inside the features folder, create the following 4 files.
    - create.js
    - read.js
    - update.js
    - delete.js
5. read.js
    - copy the content of handler.js to read.js
    - delete handler.js
